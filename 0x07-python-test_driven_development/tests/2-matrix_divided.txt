=============================================
This tests explains matrix_divided() function
=============================================

The function prints the divided matrix_divided

Usage
=====

```matrix_divided(matrix, div)``` divides the whole
matrix with the divisor

Correct inputs
==============

::
	>>> matrix_divided = __import__('2-matrix_divided').matrix_divided
	>>> matrix_divided([[12, 15], [12, 33], [34, 78]], 3)
	[[4.0, 5.0], [4.0, 11.0], [11.33, 26.0]]

Can divide regular matrix

::
	>>> matrix_divided([[1, 2.4], [2, 5], [99, 80]], 10)
	[[0.1, 0.24], [0.2, 0.5], [9.9, 8.0]]

============

::
	>>> matrix_divided(None, 9)
