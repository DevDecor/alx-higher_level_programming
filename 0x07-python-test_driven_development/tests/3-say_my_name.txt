===========================================
Say my name function ```say_my_name(...)```
===========================================

The function prints full

Usage
=====

```say_my_name(first_name, last_name="")``` prints my full name

::
	>>> say_my_name = __import__('3-say_my_name').say_my_name
	>>> say_my_name("Tiamiyu")
	My name is Tiamiyu 

Default last name used

::
	>>> say_my_name("Tiamiyu", "Abdussomad")
	My name is Tiamiyu Abdussomad
